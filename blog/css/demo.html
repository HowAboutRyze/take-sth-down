# css3 实现点点点 loading 动画

animation 的 animation-timing-function 属性除了常用的 三次贝塞尔曲线 外还有 step() 函数，该函数指定关键帧之间的跳跃，其 step-start 值是跳过 0% 进行阶跃，实现三个点逐个出现的动画。
